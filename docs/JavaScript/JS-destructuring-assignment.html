Destructuring assignment 解構賦值

在 JS 操作資料的情境中，常常會不需要整包資料都拿進來用，所以解構可以將在 Object 和 Array 中取需要的資料出來用即可，通常是透過 unpack 把資料存在另一個變數。

HOW#

  • 使用一個變數去接需要的值
  • 遇到多層的資料需要拿出來需要解到該層去
  • 如果預設值要呼叫 prompt 函數傳入，僅會針對尚未被賦值的空位置起作用
  • 物件的賦值可以不管順序，它是按照 key 去傳 value
  • 重新命名可以同時給定默認值
  • 如果只取要用的值剩餘的值可以透過 ...rest剩餘模式(pattern)去接，名稱可以任意命名

基礎語法#

// object examplelet users = {  id:1,  name:'Jeffrey',  phone: '0912-345-678',  email: 'j123@gmail.com',  title: 'front-end developer'}let { name, phone } = users; // key 要一樣let { name, phone:phoneNumber} = users; // 取別名let { name,gender='male' } = users; // 給一個新 key 加上默認值
// array examplelet [a, b = 2] = [1];
// 多層解構let users2 = {  id:1,  name:'Jeffrey',  phone: '0912-345-678',  email: 'j123@gmail.com',  title: 'front-end developer',  address:{    postal:403,    city:'Taichung',  }}
let {name,address:{ ,city}} = user2;
let [firstName, surname] = "Jeffrey Lin".split(' ');console.log(firstName, surname);
let [firstName, ,surname] = ['Jeffrey',007,'Lin'];console.log(firstName, surname);
// 處理重複值 存進去唯一值let [one, teo ,three] = new Set([1, 2, 2, 3];
// 用 entries 去循環let user = {  name: "John",  age: 30,  a:11,  b:22,};
for (let [key, value] of Object.entries(user)) {  alert(`${key}:${value}`);}
// 用 Map 循環
let user = new Map();user.set("name", "John");user.set("age", "30");
// Map 是以 [key, value] 一對的方式迭代，非常便於解構for (let [key, value] of user) {  alert(`${key}:${value}`);}
// 交換值的方式let guest = "Jane";let admin = "Pete";// 讓我們來交換變數的值：使得 guest = Pete，admin = Jane[guest, admin] = [admin, guest];
